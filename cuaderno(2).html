Eventos

Definición de evento en JS: algo que sucede y que puede capturarse para que se produzca una acción.

Referencia: https://developer.mozilla.org/en-US/docs/Web/Events

Eventos de ratón:

  • click - botón izquierdo
  • dblclick - doble click
  • contextmenu - botón derecho del ratón (muestra el menú contextual)
  • mousedown - se pulsa un botón del ratón (al presionarla)
  • mouseup - se deja de pulsar un botón del ratón (al liberarlo)
  • mouseenter - cuando el cursor entra en una zona
  • mouseout - cuando el cursor sale de una zona (fuera del elemento o encima de un elemento hijo)
  • mouseleave - cuando el cursor sale de una zona (fuera del elemento sin tener en cuenta los hijos)
  • mouseover - cuando el cursor entra en una etiqueta y también cuando entra en una etiqueta hija
  • mousemove - cuando el cursor se mueve (se repite)

Eventos de teclado:

  • keydown - se pulsa una tecla del teclado (al presionarla)
  • keyup - se deja de pulsar una tecla del teclado (al librerarla)

Otros eventos:

  • change - se produce un cambio en el control, por ejemplo se selecciona o deselecciona un checkbox o se selecciona un elemento de un select. También se produce cuando se cambia el contenido de un textbox y se cambia el foco (deja de estar seleccionado el textbox). Implica que se debe producir un cambio de foco (en este caso cambia el elemento seleccionado).
  • input - se produce un cambio en el control, pero no es necesario que pierda el foco.

Nota: En los radio button el evento change se produce cuando se selecciona el elemento, pero no cuando se deselecciona. Haz el ejercicio 5a-2 para comprobar su funcionamiento.

Nota: Las pantallas táctiles tienen su propio conjunto de eventos: touchstart, touchend, touchmove...

Evento Input vs Change

El evento input se activa cada vez que cambia el contenido del cuadro de texto.

En un lugar de la mancha

El evento change se activa al cambiar el contenido del cuadro de texto y perder el foco (cuando deja de estar seleccionado). Escribe algo debajo y pulsa Tab o haz click en alguna parte de la web.

En un lugar de la mancha
Este es el HTML asociado:
label for="txtDato">Escribe un dato:</label>
<input type="text" id="txtDato">

<p>El evento <code>input</code> se activa cada vez que cambia el contenido del cuadro de texto.</p>
<div id="divDatoEscrito" style="filter:opacity(0.5);padding:0.25rem;"></div>

<p>El evento <code>change</code> se activa al cambiar el contenido del cuadro de texto 
y perder el foco (cuando deja de estar seleccionado). Escribe algo debajo y pulsa 
<kbd>Tab</kbd> o haz click en alguna parte de la web.</p>
<div id="divDatoEscrito2" style="filter:opacity(0.5);padding:0.25rem;"></div>
Y aquí abajo está el código que tendremos que ejecutar.
const txtDato = document.querySelector("#txtDato");
	
txtDato.addEventListener("input", e=>{
  const divDatoEscrito = document.querySelector("#divDatoEscrito");
  divDatoEscrito.textContent = txtDato.value;
});

txtDato.addEventListener("change", e=>{
  const divDatoEscrito2 = document.querySelector("#divDatoEscrito2");
  divDatoEscrito2.textContent = txtDato.value;
});

Ejercicio 5a-2: Crea una web que contenga un formulario con dos radio button cuyo name sea txtOpcion:

  • colores
  • animales

Usa el evento change para que si se selecciona colores, se muestren algunos nombres de colores en alguna parte de la web, pero si se selecciona animales se mostrarán algunos nombres de animales.
Pista: añade un eventListener con change a todos los radiobutton cuyo name=txtOpcion. En el código del eventListener puedes recuperar el value del que esté :checked o directamente recuperarlo con el atributo value.

Posible solución:

Los radioButton tienen como id txtOpcionColores y txtOpcionAnimales.

Selecciona una opción


function gestionaOpciones(){
  const selección = document.querySelector("[name=txtOpcion]:checked").value;
  if(selección == "colores")
    alert("Rojo, Verde, Azul");
  else
    alert("Zorro, Perro, Gato");
}

document.querySelector("#txtOpcionColores").addEventListener("change", gestionaOpciones);

document.querySelector("#txtOpcionAnimales").addEventListener("change", gestionaOpciones);
